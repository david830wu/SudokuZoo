# SudokuZoo

SudokuZoo is a collection of varies sudokus.

## Varies Games

- 9x9 Sudoku
- 16x16 Sudoku
- Irregular house Sudoku

## Varies Solvers

- Knuth's dancing links algorithm for exact cover problem
- Backtracking algorithm
- Brute force attack algorithm

## Varies UIs

- Console
- ncurses
- Qt
